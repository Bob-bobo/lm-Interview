# 买房助手小程序

## 项目简介
一款功能全面的微信小程序，帮助购房者记录房源、分析需求、计算贷款、评估价格走势。

## 功能模块

### 核心功能
1. **🏠 房源管理** - 记录房屋价格、面积、朝向、户型、装修等完整信息
2. **🤖 AI智能推荐** - 分析用户需求，从收藏中匹配最佳房源
3. **📈 价格走势** - 记录历史价格，趋势预测和走势分析
4. **💰 房贷计算器** - 等额本息/等额本金/组合贷，完整还款明细
5. **📊 税费计算器** - 契税、增值税、个税、维修基金
6. **⚖️ 房源对比** - 多房源横向对比，综合评分排名

### 辅助工具
7. **📝 看房日记** - 记录看房感受、照片、评分
8. **🗺️ 周边配套** - 交通/教育/医疗/商业/环境五维评分
9. **📖 购房攻略** - 全流程购房指南、合同注意事项
10. **⏰ 提醒管理** - 贷款还款日、合同到期等提醒
11. **📐 面积换算** - 建筑面积/套内面积/公摊计算
12. **🎨 装修预算** - 按档次预估装修费用
13. **🧭 朝向指南** - 各朝向优缺点说明

## 技术栈
- 微信小程序原生开发（WXML + WXSS + JS）
- 本地存储（wx.setStorageSync）
- AI分析引擎（关键词匹配 + 评分算法 + 线性回归趋势预测）

## 使用说明

### 1. 准备 tabBar 图标
在 `images/tab/` 目录下放置以下图标（81×81px）：
- `home.png` / `home-active.png` - 首页
- `house.png` / `house-active.png` - 房源
- `ai.png` / `ai-active.png` - AI助手
- `calc.png` / `calc-active.png` - 计算
- `profile.png` / `profile-active.png` - 我的

### 2. 配置 AppID
在 `project.config.json` 中修改 `appid` 为你自己的小程序 AppID。

### 3. 接入后端 API（可选）
当前版本使用本地存储 + 内置 AI 分析引擎。如需接入真实 AI API：
- 修改 `utils/ai.js` 中的 `generateAIResponse` 方法
- 将关键词匹配替换为真实的 LLM API 调用
- 修改 `app.js` 中的 `globalData.baseUrl`

### 4. 微信开发者工具导入
使用微信开发者工具导入本项目目录即可预览运行。

## 注意事项
- 所有数据仅存储在用户设备本地，安全私密
- AI分析基于内置规则引擎，不依赖外部API
- 价格趋势使用简单线性回归，仅供参考
